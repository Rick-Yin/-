### 3. 遇到的Bug & 解决思路

#### 3.1 Git访问时显示22端口被禁用

**Bug原因**: 校园网图信禁用了22端口导致Git无法通过22端口进行访问
**解决思路**: 更改ssh的configuration:
```config
Host github.com
	User git
	Hostname ssh.github.com
	PreferredAuthentications publickey
	IdentityFile ~/.ssh/id_rsa
	Port 443

Host gitlab.com
	Hostname altssh.gitlab.com
	User git
	Port 443
	PreferredAuthentications publickey
	IdentityFile ~/.ssh/id_rsa
```
